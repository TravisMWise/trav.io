About

 Dimension is a 2D Platformer where the user plays through the create levels and defeat all the enemies and bosses. I have removed all the enemies from the game to make loading easier on the user and this is simply to show off key qualities the game has. Namely, the levels I created throughout the game, and the game's level editor capabilities. Things not shown here are the many enemies we created including four major bosses and the magic abilities such as lightening and fire attacks which would simply add to the load time since attacking isn't needed with no enemies.

Game Controls

  • Move: Left Arrow and Right Arrow Keys
  • Jump: Z Key
  • Basic Attack: X Key
  • Dash: Shift Key

Editor Controls

  • Toggle Editor: Escape Key
  • Toggle Collision Editor: Enter Key
