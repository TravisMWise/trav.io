How To Play

About

  This game is my first major game made using Unity Engine. The game is a Platformer style game with basic movement in which the player uses a bow and arrow to kill enemies and complete levels. The game only has three levels which are very easy to complete and I would like to expand on this in the future but before I do I want to work on other game types and gain more knowlege of Unity in general.

Game Controls

  • Move: WASD
  • Jump: Space Bar
  • Basic Attack: Left Mouse Button
  • Wall Grab: Shift or E Key

Scoring and Winning

  There is no scoring but the user can "Win" by getting to the end of the game. Later on I would like to add a boss the user can kill to finish the game and this would be the win condition.
