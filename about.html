Senior Design

  Senior Design was a full year for me and my other four group members, we chose to do Spring-Fall and skip the summer so we could all do internships over the summer. We chose to do a self-sponsorship project for the class so we would have the freedom to do whatever we wanted and not be locked into a project we did not like and did not want to do. After a few meetings about what we wanted to do and a few project ideas later we decided to create a game. Throughout the year of development this idea went through many variations and finally became a game engine, a level editor, and a game all in one project and all made from scratch using pure JavaScript and the p5 library which was only used to load images for rendering.

  The engine and level editor can be reused and extracted from the game for reuse as needed, but we had a major flaw that we could not fix by the end of our senior design class. Namely, we had an issue with loading that took us months of working with and trying to debug that never fully got solved, although we did reduce the problem dramatically. In the end the strength of the computer running the game determined how much we were able to load into the game at any point, this dramatically limited the scope of the game and the number of enemies we were able to add to the game and we had to scrap most of the work we put into the AI's simply because we could not load them all. I have a demo of the game on this website here, where you can explore the game without the enemies to ensure anyone can play the game and explore the game's graphics and map since this was a major part of my contribution to the project.

  Even though we had our issues with the project, we all learned a lot about game development and creating a major project with multiple contributors. Personally I found that I really enjoy making games and would like to spend more time making more serious games using more powerful tools such as Unity and Unreal.

Internship

  I worked at Applied Research Associates (ARA) in the Virtual Heroes Division as a Software Engineering Intern from February 2022 to November 2022. As stated above, the plan was to work an internship over the summer, but I was able to get in early and stay a bit later than expected. This was nice because I was able to get more experience and learn more about the industry, but this was very difficult for me to maintain school life with my higher-level class and an internship, not to mention working my project in senior design.

  Nevertheless, I did learn much about the industry during the internship. I worked on the frontend and backend in various projects mainly using Go, Python, and Angular. Each of which I had vary little experience with before the internship. Using Go I mainly working on creating concurrent programs to speed up programs made using python which resulted in significant speedups of 50-70%. I used python in the backend to send, receive, and modify information for the frontend of applications. Then by far my favorite part of the internship was using Angular to create, modify, and debug applications.

  I did, however, have many things I did not like about this internship which eventually led me to faze out of working at ARA. By far my biggest issue was that since they were contracted for work I never knew what I would be working on the next day. At any point in the day/project I could be told that I am being switched to a new project and each project was something I had never worked on before meaning I needed to drop everything and then do hours of research on a new topic that I had no experience with. These switches meant that I spent most of my time just learning/reading about technology instead of actually creating anything and since I was focusing mostly on school and working part time it was very hard to get things done at the internship. Another issue was the work environment, namely, the people working there did not seem to care about the company and I felt like I was back at Command Medical Products which left me with a bad feeling. Finally, I did not realize that they were government contracted before I started working there and I was working on projects for the military that I was not comfortable with (which I cannot discuss because of an NDA). All of these together and more made me know that the job was not something I wanted to do in the future and helped me learn a lot about what I did want for my future.
