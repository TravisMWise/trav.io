Stargazers

How To Play

About

 This game is a simple asteroids like game which the player controls a space ship and can fly through space avoiding asteroids. This game was made using JavaScript and WebGL which was made as a final project for my Computer Graphics class.

Controls

  • Move In Space: WASD Keys
  • Accelerate Ship: O Key
  • Decelerate Ship: P Key
  • Adjust Camera: Up/Down/Left/Right Arrow Key
  • Shoot Projectiles: Space Bar

Scoring and Winning

 You can gain score by flying through space and avoiding asteroids, the further you go the higher your score. The player has three lives and loses one with each asteroid hit. There is no end to the levels and this is a high score game which means with each time you play you try to beat your highest score achieved.
